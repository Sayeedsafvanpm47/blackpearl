OG IMAGE PLACEHOLDER
====================

For social media sharing (Open Graph), you need to create an image file at:
/public/og-image.jpg

Specifications:
- Dimensions: 1200 x 630 pixels
- Format: JPG or PNG
- Content: Include the Black Pearl Studios logo and tagline
- Background: White or light distressed paper texture
- Text: "Black Pearl - Printing, IT & AI Studio in Doha Mall"

This image will be displayed when your website is shared on:
- Facebook
- LinkedIn
- Twitter
- WhatsApp
- Other social platforms

You can use the logo from /public/logo.svg as the main element.
